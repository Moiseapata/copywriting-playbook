# Adapt Tone Prompt

**Tags:** #prompt #tone #voice #copy-editing #claude

## How to Use This Prompt

When you have copy written in one voice/tone and need to adapt it for a different audience, channel, or brand.

---

## The Prompt
Adapt the following copy from [current tone/context] to [desired tone/context].

Original Copy:
[paste copy]

Current Context: [e.g., formal sales page, casual email, technical documentation]
Desired Context: [e.g., casual email, DM outreach, video script, landing page]

Adjust for:

Reading Level: [e.g., 3rd grade for cold outreach, 8th grade for B2B]
Pronouns: [more "you" / more "we" / balance]
Sentence Length: [shorter for mobile/email, can be longer for sales pages]
Formality: [more casual / more professional]
Emotional Load: [higher for warm traffic, lower for cold]
Additional Rules:

Cut every word that doesn't earn its place
Replace abstract claims with concrete, visualizable statements (Harry Dry's Rule #1)
Make falsifiable claims where possible (Harry Dry's Rule #2)
Ensure only this brand could say it (Harry Dry's Rule #3)
Deliver the adapted copy.

---

## Reference

**Harry Dry's 3 Rules:**
1. Can I visualize it?
2. Can I falsify it?
3. Can nobody else say it?

**Suby's Consumption Principle:** Copy should look like an email from a colleague, not a promotional flyer.
