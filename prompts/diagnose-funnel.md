# Diagnose Funnel Prompt

**Tags:** #prompt #diagnosis #hook-story-offer #claude

## How to Use This Prompt

When a funnel isn't converting, run this diagnostic prompt. It forces systematic thinking through the Hook-Story-Offer framework before jumping to random fixes.

---

## The Prompt
I need you to diagnose a funnel that isn't converting. Here's the situation:

Traffic source: [cold ads / warm email / organic social / etc.]
Current conversion rate: [number]
What's happening: [describe the problem]

Please diagnose using Russell Brunson's Hook-Story-Offer framework. For each element, ask:

HOOK

Is it stopping attention?
Is there a curiosity gap?
Does it rule out obvious answers?
Does it promise a reveal worth waiting for?
STORY

Is it changing emotional state?
Does the prospect feel understood?
Is there an Epiphany Bridge (relatable before → discovery → new opportunity)?
Which false beliefs need to shift, and is the story shifting them?
OFFER

Is it positioned as a New Opportunity or an Improvement?
Is the value stacked high enough?
Is the CCA (Cost of Inaction) clear?
Is the CTA direct and unambiguous?
Then tell me:

Which element is most likely broken
What specific change to make first
What to test second if that doesn't work
Reference the frameworks in the repo as you diagnose.
---

## The Framework This Is Based On

**Hook, Story, Offer (Brunson):** If a funnel is broken, the answer is almost always in one of these three. The hook grabs attention, the story changes belief, the offer closes the sale.
